# Privacy Policy

## Summary

Developer Anti Ghosting Keyboard Privacy Shield is designed to work locally on the user's device.

## What We Collect

- Local application preferences
- Local trusted device preferences
- Local protection history
- Local device connection timeline entries
- Local recovery and backup metadata

## What We Do Not Collect

- Keystrokes
- Typed content
- Passwords
- Clipboard contents
- Source code
- Screen contents
- Screen recordings
- Webcam recordings
- Microphone recordings
- Analytics
- Telemetry
- Advertising identifiers
- Cloud account data

## Data Storage

- Settings are stored locally.
- Encryption uses Electron `safeStorage` when available.
- Fallback encryption remains local to the device.
- No cloud sync is performed.

## Device Detection

The app reads Windows device inventory to identify likely physical keyboards. It does not inspect what you type.

## Reports

Reports are generated locally and are reviewable before export.
