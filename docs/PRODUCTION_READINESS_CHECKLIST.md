# Production Readiness Checklist

## Must Have

- Stable keyboard detection for built-in and USB keyboards
- Passing lint, tests, security check, and build
- Clear privacy policy
- Support and troubleshooting documentation
- Known limitations documentation
- Release process documentation
- Accessibility QA checklist
- Security review document

## Before Public Windows Launch

- Code signing certificate in place
- Signed Windows artifacts verified
- Installer QA completed
- Portable build QA completed
- Upgrade testing completed
- Uninstall cleanup verified
- SmartScreen reputation plan prepared
- Public support workflow documented

## Recommended Validation Matrix

- Windows 11 laptop with built-in keyboard only
- Windows 11 laptop with built-in + one USB keyboard
- Windows 11 laptop with built-in + Bluetooth keyboard
- Windows 11 laptop through dock or hub
- Windows desktop with one USB keyboard
- System using high contrast and large text
- Screen-sharing or meeting app active
