# Accessibility QA

## Manual Checks

### Keyboard-Only Use

- Tab order reaches top bar, sidebar, quick actions, cards, and detail panels
- Focus indicators remain visible in dark, light, and high-contrast themes
- Dropdowns can be opened and closed without a mouse

### Screen Reader Checks

- Main navigation is announced clearly
- Buttons and menus have understandable labels
- Timeline and report sections remain readable

### Visual Checks

- High Contrast Mode remains readable
- Large Text Mode does not break key layouts
- Reduced Motion Mode removes non-essential motion
- 125%, 150%, and 200% zoom still keep major controls reachable

### Cognitive and Sensory Checks

- Simplified Mode reduces clutter
- Emergency Privacy Mode remains easy to understand
- Important actions use plain language

## Regression Targets

- Device Center trust controls
- Workspace Center summaries
- Reports Center forms
- Backup and restore flows
